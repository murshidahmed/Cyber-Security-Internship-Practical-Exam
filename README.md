# Cyber Security Internship Practical Exam

## Application Setup

OWASP Juice Shop – an intentionally vulnerable web application designed for security testing and learning purposes.

### Installation Steps:

1. Installed Node.js
2. Ran the application using:
   npm start juice-shop
3. Accessed the application via:
   http://localhost:3000

## Tools Used:

- Burp Suite
- Browser Developer Tools
- Postman

## Objective:

To identify and analyze web application vulnerabilities.

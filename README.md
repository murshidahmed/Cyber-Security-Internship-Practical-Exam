# Cyber Security Internship Practical Exam

## Application Setup

The OWASP Juice Shop vulnerable web application was used.

### Installation Steps:

1. Installed Node.js
2. Ran the application using:
   npx juice-shop
3. Accessed the application via:
   http://localhost:3000

## Tools Used:

- Burp Suite
- Browser Developer Tools
- Postman

## Objective:

To identify and analyze web application vulnerabilities.

# OWASP Juice Shop Security Assessment

## Setup Instructions

1. Clone the repository:
   git clone https://github.com/juice-shop/juice-shop.git

2. Navigate to the folder:
   cd juice-shop

3. Install dependencies:
   npm install

4. Run the application:
   npm start

5. Open in browser:
   http://localhost:3000

## Tools Used

* Burp Suite
* OWASP ZAP
* Browser Developer Tools

## Project Objective

This project identifies and analyzes common web vulnerabilities in OWASP Juice Shop and provides mitigation strategies.
